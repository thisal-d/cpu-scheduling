### **Solution to Priority Practice Question 2**

**Order by priority (lower number first):** P3 → P1 → P4 → P2

**Gantt Chart:**
`| P3 (0-7) | P1 (7-11) | P4 (11-13) | P2 (13-16) |`

**Waiting Times:**
* P3: 0 ms
* P1: 7 ms
* P4: 11 ms
* P2: 13 ms

**Turnaround Times:**
* P3: 7 ms
* P1: 11 ms
* P4: 13 ms
* P2: 16 ms

**Averages:**
* Waiting time = $7.75$ ms
* Turnaround time = $11.75$ ms
